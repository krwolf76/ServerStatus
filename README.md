This plugin is very easy, it checks the server status.

![image](https://i.imgur.com/624p9Eb.png) 

## Configuration

```json
{
  "Discord WebHook": "webhookurl",
  "Time Format": "MM/dd/yy HH:mm:ss",
  "Server Status (don't change)": false
}
```
